sum

def sum(expr: vecpy.base.Expr, deviceID=0) -> float:

Return the sum of each value in a vector expression.

Args#

  • expr : VecPy expression to sum.
  • deviceID (optional) : device ID to use.

Returns#

The sum of the expression.
