The benchmarks below show how many elements per nano-second each library can process as a function of the number of elements (nnn). Each figure shows the performance on a log-log time scale in base 10 format.

The benchmarks were conducted using:

  • Intel(R) Core(TM) i7-6700K CPU @ 4.00GHz
  • NVIDIA(R) GeForce RTX 2080 Ti
